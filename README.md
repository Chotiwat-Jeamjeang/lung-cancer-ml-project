# Lung Cancer Classification

โปรเจกต์นี้เป็น Deep Learning model สำหรับจำแนกภาพ CT Scan ว่ามีมะเร็งปอดหรือไม่

## Dataset

Dataset ที่ใช้ในโปรเจกต์นี้มาจาก Kaggle  
ดาวน์โหลดได้ที่: https://www.kaggle.com/datasets/andrewmvd/lung-and-colon-cancer-histopathological-images

> หมายเหตุ: Dataset ไม่ได้รวมอยู่ใน repo นี้ กรุณาดาวน์โหลดและวางไว้ในโฟลเดอร์ `dataset/`
